The way I think

Design before delivery.

A training event is the last thing I build, not the first. The real work happens before anyone enters the room.

See how I think about design
  1. 01

    Understand the gap

    I start with a training needs analysis — talking to SMEs and stakeholders before assuming the problem is "not enough training."

  2. 02

    Define what changes

    Clear, observable learning objectives. If I can't picture the learner doing something differently, the objective isn't done.

  3. 03

    Architect the journey

    Activities chosen for a reason — experiential, scenario-based, reflective — sequenced so each one earns the next.

  4. 04

    Build in reflection & evaluation

    Reflection isn't a closing ritual; it's where behaviour change is rehearsed. And I plan how I'll know it worked.
