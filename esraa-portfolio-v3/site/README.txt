ESRAA EBRAHIM — PORTFOLIO WEBSITE
==================================

HOW TO USE
----------
1. Open index.html in any browser to view it locally.
2. To publish: upload the whole folder to any static host
   (GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any web server).
   No build step, no dependencies — it's plain HTML/CSS/JS.
